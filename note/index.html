四十略不惑

2024-03-26 / 思考 / 随手记 /

孔子《论语·为政》：“吾十有五而志于学，三十而立，四十而不惑，五十而知天命，六十而耳顺，七十而从心所欲，不逾矩。”

这个“不惑”，不同的人理解不一样。如果从孔子本人的立场来思考，就需要从他的作品中找到相关信息进行梳理。

孔子《论语·子罕》：“知者不惑，仁者不忧，勇者不惧。”

这一段的意思是：“有智慧的人不迷惑，具有仁德的人不忧虑，勇敢的人不畏惧。”

把这个“知者不惑”与“四十而不惑”放到一起，就能知道孔子想表达的意思是他40岁的时候是一个有智慧的人。

我今年四十岁，如果按照孔子的标准，应该是“不惑”之人，一个有智慧的人。然而，我自知并不智慧，甚至还很愚钝。只是经过四十年的生命体验，对一些事情略知一二而已。

漫长的咳嗽

2023-10-06 / 随手记 /

我知道自己有慢性咽炎，也知道它无法根治，所以也接受它将一直存在于我的生活之中。对于出现咳嗽的情况，我一般选择让其自然疗愈。

最早出现歇斯底里地咳嗽是在2011年，那个时候我在长沙市蔡锷路世纪新娘婚纱摄影担任店经理。开始咳嗽到完全康复的时间尽管长，但还不算特别严重。解决方案也挺简单的，当我喉咙痒、想要咳嗽的时候，只要小口小口地喝冰可乐就能把咳嗽压下去，一个星期就能痊愈。这样的咳嗽，每年会来一次。

再往后就是2015年至2016年。我在长沙市开福区二十四节气草本生活家担任店经理的工作时，受到艾灸产生的浓烟刺激，每年会有至少2次以上的持续性咳嗽，持续时间也是一个星期左右。

2023年的咳嗽特别漫长、也特别严重。晚上会被咳嗽搞醒来，睡不着，最严重的时候感觉无法呼吸、透不过气来。这次咳嗽从9月份开始，持续了1个月，至今还没有康复。

最初只是咽喉有不适的感觉，我认为我可以挺过去，所以没有打针吃药，任由咳嗽自由发展。

暑假前暴走北京城

2023-07-07 / 随手记 /

第一次去北京是2014年8月，之后的2个月又去了2趟，那时候去北京是为了学习教练技术的课程。今年7月份，时隔9年，我和夫人带着大小宝以及岳母安排了一趟北京游。这次北京游的主要起因是夫人之前曾提出在大宝小学毕业考试之后参观下北大清华（后记：参观清华北大对于孩子的成长帮助非常有限），另外一个原因是岳母近期快要过60岁生日了，请她去北京旅游算是给她一个小小的生日礼物。

安排旅游行程并不是一件简单的事情，尤其是北京旅游安排，各主要景区或场馆需要提前预约，而去北京旅游的人特别多，预约特别困难。当预约不上相应的场馆时，需要调整安排。

景点或场馆安排

岳母想去的地方是毛主席纪念堂、天安门城楼、故宫博物院、颐和园、圆明园、八达岭长城、天安门广场升旗仪式；我想让大宝去参观的地方是清华大学、北京大学、故宫博物院、国家博物馆；小宝想去的地方是天安门广场升旗仪式、军事博物馆；夫人则认为来一趟北京很不容易，哪儿都想去看看，安排充实一点；我呢，受到炎热天气的影响，只想去有遮挡太阳的地方，比如颐和园、国家博物馆、军事博物馆。

要说关注细节和安排招待，我得向她学习。

飞哥在家休息，特地安排了一大桌子的好菜，约上公司的八九个同事陪我过生日。豆豆为此还请了半天假给飞哥帮忙，先桂也是老早就跑到飞哥家里去打下手了。我笑称这不打两次麻将送点红包给他都过意不去。

大公主和小宝也让我感到惊喜。因晚上已确定在外吃饭，于是就点了蛋糕送回了家，让夫人和孩子们一起享用。大宝在吃蛋糕之前，特地把蜡烛点起来，打了一个视频电话让我许愿、一起吹蜡烛，仪式感满满。重点是她明白这生日蛋糕的意义，不只是为了吃蛋糕。

这一整天被人关爱着，沉浸在爱的世界里，感受别人对我的关爱，幸福感满满。让我不禁思考：“如何才能让公司的每一位伙伴都能感受被爱的感觉呢？”这将是我接下来需要落实的、非常重要的一项工作。
