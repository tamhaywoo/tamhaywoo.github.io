2010-06-10

非常喜欢黑莓8320 4.2ROM的字体，周正而秀丽。不过，却遇到浏览器无法打开页面但QQ和尚邮以及证券软件都可以正常使用的状况。搜索了很久，没有人说清楚到底是怎么回事。极度纠结于4.5ROM&4.2ROM之间，近日有空，在六次刷机之后已经摸索到问题的可能之处，已经可以上网，只是有点麻烦。

可能导致问题产生的原因：

  • 安装了尚邮软件且尚邮软件一直处于在线收取状态；
  • 导入了经过修改可以更换主页地址的service book；

解决问题的方法：

  • 谨慎导入service book；
  • 使用浏览器之前将尚邮软件设置成离线状态；
  • 关闭QQ程序；
  • 如果还是不可以，重新启动手机；
  • 多做积德的事情，积累人品和运气。

2010-03-04

几天前遇到了一些不顺心的事情，就连身边的电脑、宽带都出了问题，而平时很听话的黑莓8310也开始频繁死机和反应缓慢。多次刷机后依然如故，实在没有办法了。之后搜索网上关于此类问题的话题，发现大部分都在说“2G的卡”以及“大量文件”之类的原因，所以特意将储存卡内的文件重新整理，将图片文件剪切到BLACKBERRY文件夹以外。通过这个操作，我发现之前的问题不再出现，所以以后不再将图片文件存放到BLACKBERRY目录下的PICTURE文件夹，以减少因系统扫描内存而当机的情况发生。

有人只做陌生人，有人只爱陌生人。有人只做蛋炒饭，有人只爱蛋炒饭；我想起了招商银行的广告语"因你而变"…

2010-01-10

尽管小忆和戌与能够心贴心，尽管戌与在任何时候都想着小忆，甚至包括与家人独处的时间… …但那一天最终还是来了，天气好像特别的冷，比森林里下雪的任何时候还要冷。戌与在布告栏里面看到了小忆对线团的歉意，它们都不愿意让常年关心自己的家人受到伤害，而现在这种伤害似乎已经逐渐开始，甚至已经伤害到了小忆。它明白小忆也在痛苦的矛盾之中无法自拔，甚至用“报应”和“贱”的字眼来形容自己，戌与知道此刻的它必须要承担责任。

于是戌与对小忆说：“我们来玩捉迷藏吧？我们同时跑到森林里面藏起来，等一段时间以后我们再去寻找对方，看那个时候的你我是否还能认识。”小忆当然明白这句话的意思，它说：“如果是这样，那么我将会藏起来不再让你看见，你能做到吗？”

戌与很难做这决定，但它在说了一大堆废话之后还是说了：“我们开始吧…”这句话刚一落音，小忆就飞快地跑向了森林… …戌与知道，这将是它最后一次看到小忆的背影… …

戌与喃喃地念叨：“亲爱的，我爱你！我希望你一辈子都幸福快乐！我会记得在我短短的这一辈子当中，只有你给我带来了爱情的心跳。在适当的时候我会将我们的故事说给我的家人听，说给我的孩子听… …亲爱的，对不起！…我爱你！”

那颗闪烁的水珠… …消失了… …

2010-01-09

天空灰得像哭过，离开你以后并没有更自由。酸酸的空气，守住我们的距离。一幕锥心的结局，像呼吸般无法呼吸。抽屉泛黄的日记，擦干了回忆，那笑容是傻气。你我的过去被顺时针地忘记，缺氧过后的爱情，粗心的眼泪是多余。我知道你我都没有错，只是忘了怎么退后。信誓旦旦给了承诺，却被时间扑了空。我知道我们都没有错，只是放手会比较好过，最美的爱情回忆里待续… …

第一次听周杰伦的退后，发现歌词写得很适合我的那个小小说的情节——雪原上的两只刺猬。

小忆对戌与承诺过，每当太阳照到那只猫头鹰身上的时候，它就去午睡，因为戌与不希望它变成熊猫的模样。而戌与会在小忆睡觉之后才会去睡，它觉得那是它能够体会到得一种幸福。

我想，现在的它们可能已经习惯了对方。

因为戌与每天起床都会跑到邮局去看一看，然后才拨开雪花，打开藏起来的食物。就连为麋鹿扎银针的时候都会想着小忆，最后弄得麋鹿变成了梅花鹿… …

2010-01-05

两只刺猬相识在网络上，刻意地见过一次，不经意地见过两次。如果按照网络的时间来计算，他们认识了10天；如果按照见面的时间来计算，他们只认识了10分钟。

他已经结婚了，她也快要结婚了。他们都知道什么可以做，什么不可以做。

他只要想到她就会心跳得厉害，他知道可能自己很喜欢她，或许那就是久违的恋爱感觉。他总是想着她怎么还没有出现在视线中，会不会是出了什么事情，为此很是担心，心神不宁。

她很害怕他只是将自己作为空虚的玩物，他很害怕她会这么认为，也害怕彼此靠得越近越容易伤害到对方，因为他们是刺猬，身上有刺。相对于害怕她的误解，其实他更害怕自己伤害到她，他害怕和她的跨空间情感交流影响到她的家庭。所以他在和她交流的过程中一直思考着怎样才好，因为他很不希望失去她，可越是考虑太多越是发生了他不愿意发生的事情，她觉得他仅仅是因为空虚寂寞太久… …

她离他远去，在雪地里留下一串长长的脚印，不过很快被飞雪给掩盖掉，再也找不到寻找她的痕迹。而他，伤感地望着她远去的背影，幸福地笑着，因为这也许这是保存完美心跳的最好方式。他心酸地笑着，拔下了身上的刺在石头上刻画着…因为他知道他会老去，记忆也会慢慢地衰老，所以他必须要将这份记忆保存下来，让自己永远记得，记得这份暧昧的心跳感觉。

许久以后，人们发现有一块特别的石头，上面有一个心酸浪漫的小故事，而在故事的结尾，人们得到了如意的答案——他和她彼此生活得很幸福。虽然他们不再见面，但他们仍然保持着联络，保持着适当的距离，以保持那份幸福不会轻易流失… …
