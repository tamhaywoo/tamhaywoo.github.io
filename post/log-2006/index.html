2006年随笔合集

2006-01-09 # 随笔

2006-01-09

2005年过得比以前任何一年都要快许多，发生的事情也挺多的，庆幸的是很多事情都在慢慢的往好的方向发展。

只是我还没有去见拜见她的父母，而她也没去我家。不过，这并不能造成什么坏的影响。

今年在公司里做琐碎事情比较多，但没有给自己的部门带来什么盈利，也未曾看到明显的部门成绩，这是我最惭愧的。所以每次见到那几位老总都觉得自己很不好意思，无颜相见似的... ...

做得最多的琐碎事莫过于去外展点做蜂王[1]，公司的外展活动我基本上都得出去帮忙，所以跟大学生在一起的机会也就多了很多。也常常被这些还未进入社会单纯的他们所感染，发觉每个人都是那么热情洋溢，即便是有困难也能够相互帮助以完成预定的任务。精神难能可贵，有时候也会让我振奋不已。

另外，第一次听到一些公司的销售经理说我的作品很受顾客欢迎，这是我最高兴的事情了。所以每次他们点名要我帮忙做活动用的新样品时，总会带着那份幸福及压力去认真做好。

在婚纱摄影设计方面的改变主要是摒弃了以往以构图为重点的思想，将精力放在提升原照片的色彩及氛围上，同时将所要设计的框类产品的本身色彩考虑了进去，而构图则次之... ...

摄影方面还是没有进步，一直使用松下FX5这款傻瓜数码机，一般是拍些记录生活的片子，不过也有一些还不错的照片就用做素材了。这个照片还是要坚持拍的... ...

下半年明显的感觉到身体不如从前，所以每个星期都会去打打羽毛球，未曾进步，但手臂倒是粗了不少。体重也在望期望的目标上升... ...

忽然被很多东西给堵塞住了，想说但理不清头绪，罢了，不说了。认真踏实的做人做事吧...

2006-04-17

在新天宾馆参加了亚刚和丽华的婚礼喜筵，看到了新郎被主持人整得趴在地上求婚(_ _)新郎二话没说立刻照办，可想而知他们的爱情多么甜蜜啊！

后来在喝酒的时候，有些人把餐桌上的各种油汤倒在酒杯里，给新郎敬酒时让新郎喝掉。唉...害得人家不能喝完而糗了。

话说在前头，要是哪小子在我结婚时这样子整我，我跟他没完... ...

2006-04-17

下午收到久违的柠檬丽都寄来的一封信，看完信件的内容之后心中百感交集啊... ...尤其是看到里面居然有一张签署日期为2005年9月20日的交房通知更是让我的血液流动速度加快，那个气啊... ...真的叫不是个滋味。其他的不必多说了，说了也没意义... ...

不过还是要表扬下他们还记得提醒业主去收房，至少他们还记得需要我们收房后才会有钱进帐。但是，如果柠檬丽都卖出去的那些问题房子不整改好的话，相信没有几个人会收房，这一点是可以肯定的。

2006-05-16

近两三个月可能是要够我忙的了。因为房子的设计图已经出稿，资金也筹备到位，就要开始装修我的新房了，现在只是在等我的装修委托人将剩余的具体图纸以及将他的施工队搞定。

在这里得感谢前期帮我设计的刘静、朝云哥以及汤哥，同时也必须要感谢给予我资金援助的几位大哥[2]还有一位姐姐，谢谢你们的帮助。没有你们的信任与帮助，我将一事无成... ...

2006-05-20

我向来都不奉行啃骨头，因为了解失败的痛苦才不去碰失败的钉子，所以我自小以来都是知难而退。这可以说是一种优点，因为这样的生活比较平稳；但同时也是一种缺点，因为在回避困难的同时就会失去了许多学习解决问题的机会，那么在之后遇到相同的问题时，又得掉头... ...

很庆幸自己能认识到这个问题，更庆幸自己在最近这一阵子遇到困难时并没有逃避，而是慢慢的解决问题。特此记录... ...

2006-06-03

其实很多人会对自己喜欢的事物会有所期望，期望它能够发展到自己想象中的地步，就如想要实现自己的愿望一般。但实际上却很少能够令自己称心如意，就因为这种不确定性和未知性，我们会在某一天有所期望，也会在某一天有所淡忘。

花费很多心思和时间去做的事情并不一定会让自己心身愉悦，就算一直坚持也总会有一天感觉到那是一种无形的压力和束缚。某一天的心血来潮也会被遇到的问题所冷却，如今的一切都已经冷却了，轻松也随之而来，希望这份轻松不是空虚的表面现象，但愿吧！

2006-10-15

很多事情有两种选择，以种选择会让你向上，另一种选择会让你往下。但这两者或许是有联系的，因为经历很多之后也许又会相交，只是多绕了几圈而已。这是乐观的想法，因为我不想对一切失去希望。

四年前的选择让我拥有了自己的另一半以及温暖的家，也让自己的审美以及专业技能得到了提升，人生观也发生了变化。但却让我困在现在的这个节骨眼上，我感到非常的恐惧，但也在那惊惶的思绪中保留着希望以及奋斗的位置。

现在我又将面临着选择... ...

2006-11-01

这几天一直都没有更新网站，感觉非常轻松，倒是难为了那些失落而归的朋友们。我想有些朋友不仅仅是因为这个原因而失落（或者应该把失落换成失望一词），确切的说应该是您的失望我的失落。因为很多原因，梦想中的计划并没有实现，电子杂志方面也已经悄然的停滞，因为没有好好的规划，也因为没有好好的沟通，这些问题都应该记住，避免再次发生！

公司的工作方面似乎有了很大的变化，我也期望在公司的这次变化中发生变化。我现在最恐惧的就是止步不前，所以在一阵躁动之后又沉下心来静静地等待着，也在等待中筹备着... ...希望一切如我所愿！

从明天开始就要开展为期一个月左右的培训工作了，对于这个这不是我的本职工作的项目，我倒是投入了十二分的热情在里面。因为这也算是一种变化，也是良性的，值得自己去尝试。

要充满希望的对待每一天，保持良好的心态... ...

2006-11-25

昨天在公交车上发生了一件事情，让人很不愉快，也让我再次感受到现在的人冷漠的心，而这一切都是由一个没有盖好的车顶通风窗而引起的。

昨晚的长沙依旧是大雨连连，温度也降到了十度以下。晚上十点的时候，人们都已经早早地回到了温暖的家，享受着那暖手的热茶，而我却才下班。一挤上车就发现公交车内站满了人，唯独有一块地盘是空着的，只是那里湿漉漉的，那个地方的上方是一个没有关好的车顶通风窗口，并不时地倾泻着从上帝那里泼下来的洗脚水。我想，这个车窗应该很容易关掉吧，于是伸手上前，希望能了解这个乱糟糟的事情。可是，三秒钟以后发生的事情让我很尴尬，因为这个车窗盖没有想象中的那么灵活，我就像个长臂猿一样的吊在了那里，但车顶的通风窗依旧纹丝不动，人群中传来阵阵的笑声，我想我已经成为了笑料... ...呵呵...能带来一点快乐的气氛还不错！

但我想如果几个小伙子一起来关闭它应该是没有问题的，于是我向人群申援，希望能给有人和我一起把这个盖子关掉，以便车厢内不那么拥挤，让人们多一处容脚之地。可结果让我很懊恼，因为没有一个男士愿意到雨中来和我一起把车窗关掉，只有一个女孩子挤到了我的跟前，说要和我一起关掉车窗... ...

我忽然感觉所有的人都站在没有车顶的车箱内，任由上帝肆意地向他们身上泼撒赃物，他们只是木然地站在那里，一直站着... ...赃物带来的疾病在人群中一个接连一个的传染着，没有人愿意帮助身边的人，直到他们自己变成那个祈求别人帮助的那个人。

希望这一切能够早日结束，我只想让那些人觉醒... ...
